Personal Implementation of the paper: [SAT-SKYLINES: 3D Building Generation from Satellite Imagery and Coarse Geometric Priors](https://arxiv.org/pdf/2508.18531).


## Dependencies
This repo is forked from [TRELLIS](https://github.com/microsoft/TRELLIS?tab=readme-ov-file). Please refer to the original documentation for environment setup.

## Scope
This repo aims to replicate the **Coarse Geometric Control (Sec 3.1)** feature from SAT-SKYLINES. Please communicate with the authors for the **SKYLINES-50K** dataset.
